# Weather App

This Weather App is a simple React application that fetches real-time weather data from the WeatherAPI and displays it for a specified city. Users can input the name of the city and get the current temperature, feel temperature, humidity, and wind speed.

## Features

- **Real-Time Weather Data:** Fetches current weather data from the WeatherAPI based on the specified city.
- **Dynamic Updates:** The UI updates dynamically with the latest weather information for the chosen city.
- **Input City:** Allows users to input the name of the city to fetch weather data for that location.
- **Responsive Design:** Designed with responsiveness in mind for optimal viewing across various devices.

## Technologies Used

- **React:** The application is built using React, a JavaScript library for building user interfaces.
- **useState:** Utilizes the `useState` hook to manage state within functional components.
- **useEffect:** Utilizes the `useEffect` hook to perform side effects in functional components.
- **fetch API:** Fetches data asynchronously from the WeatherAPI using the `fetch` API.
- **WeatherAPI:** Retrieves real-time weather data from the WeatherAPI to display on the application.

## Usage

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install dependencies using `npm install`.
4. Start the development server with `npm start`.
5. Access the application in your browser at [http://localhost:3000](http://localhost:3000).
